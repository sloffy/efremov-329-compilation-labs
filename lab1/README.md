# Preprocessor Lab

## Описание

В рамках лабораторной работы реализуется модуль препроцессора, предназначенный для предварительной обработки исходного кода программы. Препроцессор очищает исходный файл от лишней информации перед дальнейшим анализом.

## Цель работы

Разработать программу, которая с использованием регулярных выражений выполняет очистку исходного кода.

## Тестовая программа

Для тестирования работы препроцессора используется файл `test.cs`, содержащий различные синтаксические конструкции языка:

* объявление переменных
* присваивания
* арифметические и логические выражения
* условные операторы `if-else`
* циклы `for` и `while`
* объявление и вызов функций
* однострочные и многострочные комментарии
* лишние пробелы и пустые строки

Также в программе может присутствовать ошибка (например, незакрытый многострочный комментарий) для проверки обработки ошибок.

## Реализованный препроцессор

Препроцессор написан на языке **Java** и выполняет следующие действия:

1. Читает исходный файл с кодом.
2. Удаляет однострочные комментарии (`//`).
3. Удаляет многострочные комментарии (`/* ... */`).
4. Удаляет лишние пробелы в начале и конце строк.
5. Удаляет пустые строки.
6. Проверяет наличие незакрытых многострочных комментариев.

## Результат работы

После обработки программа выводит очищенный исходный код, а также информационные сообщения или сообщения об ошибках.

## Запуск программы

Компиляция:

```
javac Preprocessor.java
```

Запуск:

```
java Preprocessor
```

Входной файл для обработки — `test.cs`.
